xarfutil - pyxarf sample implementation
---------------------------------------

`xarfutil` is a small command line tool, which implements :mod:`pyxarf`. Using
`xarfutil`, you can create validated and compliant X-ARF reports by command line
parameters or input files. It's also possible to directly send `X-ARF
specification <https://github.com/abusix/xarf-specification/blob/master/
xarf-specification_0.2.md>`_ compliant via email.
