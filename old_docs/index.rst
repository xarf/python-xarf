Introduction
^^^^^^^^^^^^

`pyxarf` is a `Python <http://www.python.org>`_ library for handling `X-ARF <http://www.x-arf.org/>`_ Network Abuse Reporting.

* :mod:`pyxarf` : A module for creating, validating and serializing X-ARF objects.
* :mod:`xarfmail` : A module for sending X-ARF reports by E-Mail, with automatic Abuse Contact lookup provided by the free `querycontacts <https://pypi.python.org/pypi/querycontacts/>`_ library.
* `xarfutil </xarfutil>`_ : A command line client for reporting in X-ARF directly from the Shell.

For more information on the reporting format X-ARF, check out it's `offical website <http://www.x-arf.org>`_.
